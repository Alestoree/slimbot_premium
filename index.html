📦 Guía de instalación

Instalar en Railway

1Crear cuenta en GitHub

Ve a github.com → Sign up → crea tu cuenta gratis.

2Subir los archivos

  • Crea un repositorio nuevo llamado slimbot-premium
  • Sube todos los archivos de esta carpeta
  • Importante: mantén la estructura de carpetas igual

3Crear cuenta en Railway

Ve a railway.app → Login with GitHub → conecta tu cuenta.

4Crear proyecto en Railway

  • Toca New Project
  • Elige Deploy from GitHub repo
  • Selecciona tu repo slimbot-premium
  • Railway lo detecta automático y despliega

5Obtener tu URL pública

  • En Railway ve a tu proyecto → Settings → Networking
  • Toca Generate Domain
  • Te da una URL tipo: slimbot.up.railway.app
  • Esa URL es tu panel web publicado 🎉

6Conectar el bot WhatsApp

  • Ve a tu URL → Login como Admin
  • Toca el botón 📱 QR Bot arriba
  • Escanea el QR con WhatsApp → Dispositivos vinculados
  • ¡Bot conectado y funcionando 24/7!

7Generar y activar keys

  • Panel Admin → Generar Key → escribe nombre + elige plan
  • La key se crea automáticamente en la base de datos del servidor
  • Mándale la key al cliente
  • Cliente escribe: .key Lic-slim-XXXXXXXX
  • ✅ Bot activo con su licencia
